Это проект для тестирования CI/CD

Предварительный этап
1. Я создал виртуальную машину в Яндекс.Облаке
2. Купил домен https://cicd-test-project-vue.ru на регру
3. Скачал термиус

Основной этап
1. Создал репозиторий на GitHubв дальнейшем для работы с Github Actions (для настройки CI/CD в Gitlab надо зарубежную карту, а у меня нет такой)
2. Создал фронтовый проект на Vue 3 с использованием Vite
3. Изучал докер и докер-композ (написание образов)
4. Пошагово итеративно изучал и создавал разные образы, дорабатывал их, потом добавил nginx для статических файлов
4. Ручками выгружал в виртуалку репозиторий с образом, чтоб развернуть на виртуалке в докере при помощи docker compose одной командой
5. Пошагово настраивал Github Actions, создавал workflow для сборки образа и развертывания на виртуалке при помощи ssh
6. Когда все джобы начали успешно проходить я настроил SSL LetsEncrypt для домена с помощью certbot на python
